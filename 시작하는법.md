# 🚀 legendary-blacksmith 시작하는 법

## 1단계 - 의존성 설치 (최초 1회)

이 폴더(legendary-blacksmith)를 열고 터미널에서:

```
npm install
```

## 2단계 - 로컬에서 실행

```
npm run dev
```

브라우저에서 http://localhost:5173/legendary-blacksmith/ 열면 게임 실행됨

## 3단계 - GitHub에 올리기

GitHub Desktop에서:
1. "legendary-blacksmith" 폴더 전체를 레포지토리로 추가
2. Commit to main 클릭
3. Push origin 클릭
→ 자동으로 GitHub Pages에 배포됨

## GitHub Pages 설정 (최초 1회)

1. github.com에서 레포지토리 열기
2. Settings → Pages
3. Source: GitHub Actions 선택
4. 저장

## 게임 플레이

- 게임 시작 → 영지 허브
- 대장간 클릭 → 장비 선택
- 강화 클릭 → 강화 시도
- 신성력 보조 체크박스로 확률 상승/파괴 방지 선택 가능
- 턴 종료 버튼으로 다음 주로 진행
